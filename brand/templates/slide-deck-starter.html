OMNI OS · Phase 0 03

Privacy by construction.

The mesh protocol enforces privacy cryptographically — not by policy. A non-compliant node cannot produce valid traffic.

  • PII tokenization at the OS API level
  • Format-preserving encryption for routing metadata
  • zk-SNARK compliance proofs on every payload
  • TEE-bound decryption envelopes — mesh participation requires attestable hardware
